Simulation Archive

다양한 시뮬레이션을
폴더처럼 탐색하는 공간

이 사이트는 여러 주제의 시뮬레이션을 한곳에 모아둔 아카이브입니다. 메인에서는 topic 폴더들을 보고, topic 안으로 들어가면 그 안의 simulation 목록이 다시 나타나는 구조입니다. Finder처럼 직관적으로 탐색할 수 있도록 단순하고 깔끔한 인터페이스로 구성했습니다.

구조

main 전체 topic 폴더를 보여주는 시작 화면
topic 해당 주제 안의 simulation 폴더 목록
simulation 실제 시뮬레이션 페이지

주제 폴더

폴더를 클릭하면 해당 topic 페이지로 이동하도록 구성할 수 있습니다.

Finder 스타일을 참고한 메인 페이지입니다.
